# Word for SCE - מחולל מסמכי Word

מערכת ווב ליצירת מסמכי Word מעוצבים עבור המכללה האקדמית להנדסה שמעון (SCE).
האפליקציה מאפשרת למשתמשים למלא טופס עם פרטי המסמך (כותרת, סטודנטים, מנחים, מוסדות, מילות מפתח ותוכן) ולייצר קובץ `.doc` מפורמט ומוכן להגשה.

## תכונות עיקריות

- **תמיכה דו-לשונית** - עברית (RTL) ואנגלית (LTR) עם פונטים ועיצוב מותאמים לכל שפה
- **ניהול סטודנטים** - הוספת מספר סטודנטים עם כתובות אימייל
- **ניהול מנחים ומוסדות** - תמיכה במנחים מרובים ממוסדות שונים, כולל רשימה מוגדרת מראש של מרצי SCE
- **קוד מחלקה** - הוספת קוד מחלקתי (לדוגמה SE-A-20) למסמך
- **מילות מפתח** - הוספה ומיון אלפביתי אוטומטי
- **הגבלת מילים** - 120 מילים באנגלית, 200 בעברית
- **שמירה אוטומטית** - שמירת נתוני הטופס ב-localStorage עם debounce של 2 שניות
- **עיצוב כותרת** - אפשרות להחיל Title Case או לשמור על הפורמט המקורי למילים בודדות
- **ולידציה בזמן אמת** - עם הדגשת שגיאות והנחיות למשתמש

## טכנולוגיות

- **React 19** + **TypeScript**
- **Vite** - כלי בנייה עם HMR
- **CSS3** - עם Custom Properties, עיצוב רספונסיבי ותמיכה בניידים

## התקנה והרצה

```bash
npm install
npm run dev
```

לבנייה לפרודקשן:

```bash
npm run build
```

## מבנה הפרויקט

```
src/
├── main.tsx                    # נקודת כניסה
├── App.tsx                     # קומפוננטת שורש
├── components/
│   └── DocumentGenerator.tsx   # הקומפוננטה הראשית - טופס ויצירת מסמך
├── styles/
│   ├── App.css
│   └── DocumentGenerator.css   # עיצוב הטופס והממשק
└── index.css                   # סגנונות גלובליים
```

## איך זה עובד

1. המשתמש בוחר שפה (עברית/אנגלית)
2. ממלא את פרטי הטופס - כותרת, סטודנטים, מנחים, מוסדות, מילות מפתח ותוכן
3. המערכת מבצעת ולידציה על הקלט
4. בלחיצה על "Generate Document" נוצר קובץ HTML מעוצב שמיוצא כקובץ `.doc`
5. הקובץ מוריד אוטומטית לדפדפן
